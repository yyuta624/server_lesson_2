*ディレクトリ
PCのフォルダのこと。プログラミング文脈ではディレクトリを使用する。

*ls
ディレクトリ内に存在するディレクトリやファイルを一覧表示するコマンド

*pwd
現在のワーキングディレクトリを表示

*cd
指定したディレクトリに変更

*mkdir
ディレクトリを作成

*touch
ファイルの作成

*&&
and演算子
左辺の実行が成功すれば右辺を実行する

*cp
ファイルやディレクトリをコピーする

*mv
ファイルやディレクトリを移動する

*rm
ファイルやディレクトリを削除する　ディレクトリ削除の場合は-rを追加

*ワイルドガード
「*」以下の部分を対象としてコマンドを実行できる

*cat
指定したファイルを表示する

*>
左辺の実行結果の出力を右辺のファイルに書き込む

*>>
 左辺の実行出力結果を右辺のファイルの末尾に書き込む

*vi
ターミナル上で即時起動できるエディタ
i       インサートモードに切り替え
escキー ノーマルモードに切り替え
j       下に移動
k       上に移動
h       左に移動
l       右に移動
:w      保存
:wq     保存して終了
:q!     保存せずに強制終了
u       変更内容を一つもとに戻す
Ctrl + r        変更内容を一つ前にすすめる
